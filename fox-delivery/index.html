Protect the restaurant's margins in three ways

  1. Pass on more of the delivery fee to the customer. Edit the End User Delivery Fee column above to see what the take rate looks like at each distance. The sheet defaults to a 50/50 split - try nudging it upward.
  2. Set a minimum AOV for delivery. A bigger basket spreads the same delivery cost - push AOV up in the inputs and watch the take rate drop sharply.
  3. Mark up D2C prices to match marketplaces. If the restaurant is already marking up on Just Eat / Deliveroo, why not on their own Flipdish ordering channel?
